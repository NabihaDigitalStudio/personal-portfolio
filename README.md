# 🌐 Personal Portfolio Website

Welcome to my **Personal Portfolio Website** project! This is a multi-page portfolio website created using **HTML only** as part of my Web Development assignment.

## 📌 Project Overview

This project showcases my personal information, education, experience, hobbies, strengths, weaknesses, and future goals through multiple connected HTML pages.

## 📄 Pages Included

- 🏠 Home
- 👩 About Me
- 🎓 Education
- 💼 Experience
- 🎯 Hobbies
- 💪 Strengths
- ⚠️ Weaknesses
- 🚀 Future Goals

## 🛠️ Technologies Used

- HTML5
- Semantic HTML Elements
- Tables
- Lists
- Images
- Navigation Links

> **Note:** This project is built using **HTML only**. No CSS, Bootstrap, or JavaScript has been used.
